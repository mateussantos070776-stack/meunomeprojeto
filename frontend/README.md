# Frontend

Este é um scaffold minimal de frontend usando Vite + React + Tailwind + Lucide-React.

Instalação rápida:

```bash
cd frontend
npm install
npx tailwindcss init -p
npm run dev
```

Observações:
- O `tailwind.config.cjs` já existe com paths configurados.
- O projeto usa `lucide-react` para ícones.
